# AssemblyScript Test
Just some tests with AssemblyScript and AssemblyScript Loader.

## Installing

```
git clone https://github.com/Feavy/AssemblyScript-Test.git
cd AssemblyScript-Test
npm install
```

## Building

```
npm run asbuild
```

## Running

```
python -m http.server
```

Then go to `http://localhost:8000`
